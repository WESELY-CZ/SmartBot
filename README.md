# SmartBot

A simple Slack bot running on Node.js using the Bolt framework and Socket Mode.

## Features

* `/smartbot-ping` - Check the bot's latency.
* `/smartbot-help` - Show all available commands.
* `/smartbot-catfact` - Get a random fun fact about cats.
* `/smartbot-dogfact` - Get a random fun fact about dogs.
* `/smartbot-catpicture` - Get a random cute picture of a cat.
* `/smartbot-duckpicture` - Get a random picture or GIF of a duck.
* `/smartbot-joke` - Get a random joke.
